The following are some of the data types taught:

  • String

    • These are alphanumeric characters usually enclosed in quoatation marks.
    For example:"John","Nairobi" and "Peter"

  • Integer(int)

    • These are numerical values without decimals.
    For example:1, 2, 1000

  • Float

    • These are numerical values containing decimal numbers.
    For example:34.5, 24.76 and 87.355

  • Booleans(bool)

    • These are true or false values:are question-like in form.
    For example:"Yes" or "No"

  • Characters

    • These are singly inputed characters
